GitHub

  1. Create and use a repository
  2. Start and manage a new branch
  3. Make changes to a file and push them to GitHub as commits
  4. Open and merge a pull request

Creating a repository

  1. In the upper-right corner of any page, use the drop-down menu, and select New repository
  2. In the Repository name box, enter filename.
  3. In the Description box, write a short description.
  4. Select Add a README file.
  5. Select whether your repository will be Public or Private.
  6. Click Create repository.

Create a local git repository

  1. Create a local git repository -- mkdir myproject
  2. Add a new file to the repo -- touch mnelson.txt
  3. git status
  4. Add a file to the staging environment -- git add .
  5. git status
  6. Create a commit -- git commit -m "This is my first commit!"
  7. git log // git log --oneline
